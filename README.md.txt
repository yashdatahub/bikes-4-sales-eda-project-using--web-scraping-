# 🏍️ Used Bike Market Analysis (EDA Project)

## 📌 Project Overview

This project focuses on analyzing the **used bike market dataset** collected from *Bike4Sale.in*.
The goal is to understand **price trends, brand performance, and resale patterns** using Exploratory Data Analysis (EDA).

---

## 🎯 Objectives

* Analyze price variations in the used bike market
* Understand how **model year affects resale price**
* Compare **brand-wise and city-wise listings**
* Provide insights for **buyers and sellers**

---

## 📊 Dataset Description

The dataset was collected using **web scraping** techniques.

### 📁 Features (Columns)

| Column Name       | Description                                   |
| ----------------- | --------------------------------------------- |
| Brand Origin      | Country where the bike brand originates       |
| Company           | Manufacturer of the bike (e.g., Honda, Bajaj) |
| Bike Model        | Name/model of the bike                        |
| Model Year        | Manufacturing year of the bike                |
| Price             | Selling price of the bike (₹)                 |
| City              | City where the bike is listed                 |
| State             | State where the bike is located               |
| Finance Available | Whether EMI/loan option is available (Yes/No) |

---

## 📈 Key Insights

* 📌 Newer bikes (2018–2024) have **higher resale value**
* 📌 Royal Enfield & Bajaj dominate the market (~36%)
* 📌 Top 4 brands make up **60%+ of listings**
* 📌 Metro cities like Bangalore, Mumbai, Pune lead listings
* 📌 Electric bikes (Ola S1 Pro) show rising demand
* 📌 Most bikes fall in **mid-price range (₹6k–₹80k)**

---

## 💡 Business Insights

* Sellers can price bikes based on **model year trends**
* Buyers can identify **fair price range**
* Companies can target **high-demand cities & brands**
